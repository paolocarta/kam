## kam webhook

Manage Git repository webhooks

### Synopsis

Add/Delete/list Git repository webhooks that trigger CI/CD pipeline runs.

```
kam webhook [flags]
```

### Examples

```
kam webhook
create
delete
list

  See sub-commands individually for more examples
```
